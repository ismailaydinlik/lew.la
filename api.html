lew.laapi stuff
GET /api/useragent
Returns your browser User Agent. Example Response 
{
    "useragent": "Mozilla/5.0 (Macintosh; Intel Mac OS X 10_11_4) AppleWebKit/601.5"
}
GET /api/word
Returns a random English word. Example Response 
[
    "radios"
]
GET /api/word/[count]
Returns a given amount of random English words (max 100 per request). Example Response 
[
    "astronomy", 
    "philosophy", 
    "shipping", 
    "mic", 
    "shell"
]
GET /api/steam/get_player_count/[appid]
Returns the amount of players playing a given Steam game. Example Response 
{
    "gameid": 730, 
    "player_count": 791898
}
